====================================Sales Analytics Dashboard (2024)==================================================

=>Project Overview
This project analyzes company sales data using Excel, SQL, Python, and Power BI to generate actionable business insights.

1. Tools & Technologies
- Excel (data cleaning)
- Oracle SQL (data storage & validation)
- Python (Pandas, NumPy, Matplotlib)
- Power BI (interactive dashboard)

2. Project Structure
- `data/` → Sales dataset (CSV)
- `python/` → Data analysis scripts
- `powerbi/` → Power BI dashboard file
- `screenshots/` → Dashboard images

3. Key Insights
- Identified top-performing regions based on sales
- Analyzed category-wise profit contribution
- Observed monthly sales trends
- Built interactive filters using slicers
